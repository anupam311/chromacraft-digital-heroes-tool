# 🎨 ChromaCraft

A modern React-based color palette generator that helps designers and developers create harmonious color schemes with real-time customization, editing, and export capabilities.

---

## 🖥️ Live Demo
[Live Demo](https://chromacraft-digital-heroes-tool.vercel.app/)
---

## ✨ Features

- 🎨 Generate beautiful color palettes
- 🌈 Multiple harmony modes
  - Random
  - Analogous
  - Monochromatic
  - Complementary
  - Triadic
- 🔒 Lock colors while regenerating palettes
- 🎛️ Fine-tune colors using HSL sliders
- ✏️ Edit colors directly using HEX values
- 📋 Copy HEX codes to clipboard
- 📤 Export palettes in multiple formats
  - HEX
  - CSS Variables
  - JSON
- ⌨️ Keyboard shortcut (Space) for quick palette generation
- 📱 Responsive interface
- ✨ Modern glassmorphism-inspired UI

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/anupam311/chromacraft.git
```

Move into the project

```bash
cd chromacraft
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
